The **Intelligence** tab lists every observation Conway saved during the run for use by future runs. An intelligence item is a small, durable piece of context — a note about an entity, a baseline value, or a pattern worth remembering.

## Why intelligence exists

Conway gets smarter about your data over time specifically because of intelligence. A second run that touches the same account benefits from what the first run already learned — without intelligence, every run starts from zero. Intelligence isn't edits to the source data, and it isn't rules; it informs the agent's reasoning the next time it runs.

## The intelligence card

Each intelligence card shows the subject (the entity or topic it attaches to), the observation the agent saved, and a link to the investigation that produced it. Click through to read the original report and see the evidence that prompted Conway to remember this.

## Where intelligence appears

Intelligence doesn't only live here. The same observation shows up on the relevant entity's dossier in [Intel](/detection/intel), gets cited in future runs' reports when the agent uses it, and can be summoned in chat — *"what do we know about this account?"* works as a question.

## Removing an intelligence item

If an intelligence item is wrong or no longer useful, open the card and use its remove control. You can also ask the assistant directly — *"forget what you remembered about X"* — see [Chat intelligence](/chat/memories).
