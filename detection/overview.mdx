**Detection** is Conway's main hub. It's the first screen you see after logging in and the place you spend most of your time when your focus is finding things rather than working cases.

<Frame>
  <img src="/images/detection/overview.png" alt="The Detection hub" />
</Frame>

Detection has four tabs across the top, each giving a different lens on the same underlying work.

## Which tab to use

| You want to… | Start in… |
| --- | --- |
| Launch a new run | **[Runs](/detection/runs)** — click **DRAFT RUN** |
| See what a specific run found | **[Runs](/detection/runs)** — click the run |
| Catch up on what the agent flagged this week | **[Reports](/detection/reports)** — filter by verdict |
| Adopt or reject a rule the agent proposed | **[Rules](/detection/rules)** — the **PROPOSALS** sub-tab |
| Audit which deployed rules are still healthy | **[Rules](/detection/rules)** — the **MONITOR** sub-tab |
| Read the dossier on a suspicious account | **[Intel](/detection/intel)** — search for the entity |

The two counters at the top of the page (**Total Runs** and **Reports Written Daily**) update live as runs complete, so you can glance at the hub and tell whether things are moving.

<Note>
Pilot deployments do not continuously monitor rules. The **MONITOR** sub-tab shows what rule monitoring looks like but the data is illustrative. See [Pilot limitations](/pilot/limitations).
</Note>
