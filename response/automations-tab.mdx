Inside **Response**, the **Automations** tab shows the playbooks your team has set up. A playbook is a reusable recipe that fires in response to a case event or on a schedule — for example, *"when a case moves to Escalated, run the credential-check playbook."*

<Note>
**Response (including this tab) is not available in Pilot deployments.** If you are here to *build* a playbook from scratch, go to [Building a playbook](/automations/building). If you are a Pilot user, your playbooks were configured for you by the Conway team — see [Playbooks configured for you](/pilot/playbooks-configured).
</Note>

## The playbook list

Each row is one playbook. The **Live / Draft** indicator tells you at a glance whether it's firing on real cases or still a work in progress. Click any row to open the playbook in the editor ([Building a playbook](/automations/building)).

Each row also carries a short outcome breakdown over a selectable window. Toggle **10d / 30d / 90d** above the list to widen or narrow that window. Outcome pills tell you what the playbook's recent runs produced:

- **Fraud** — the run concluded the activity was of concern.
- **Legit** — the run concluded the activity was benign.
- **Escalated** — the run escalated rather than resolving.
- **Routing…** / **Processing…** — in-flight.

A playbook producing mostly *Escalated* when you expected it to resolve is worth a look — drag it into chat to ask why.

## Starting a new playbook

Use **CREATE** to build one from scratch, or **GENERATE** to have Conway propose a playbook from a prompt. Both land you in the [playbook builder](/automations/building); **GENERATE** pre-fills the builder with a draft you can then tune. See [AI-generated playbooks](/automations/ai-generated).

## Relationship to the Automations section

This tab is the *live operational view* — which playbooks exist, which are firing, how they're performing. The [Automations section](/automations/overview) of this guide covers the underlying concept and how to author or generate playbooks in the first place.
